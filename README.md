# AI-Test Repository

This repository contains a simple greeting application created in response to GitHub issue #1.

## Features

- Basic greeting functionality
- Simple command-line interface
- Unit tests for core functionality

## Installation

```bash
# Clone the repository
git clone https://github.com/mbailey/ai-test.git
cd ai-test

# Install dependencies
npm install
```

## Usage

```bash
# Run the application
node index.js
```

## Testing

```bash
# Run tests
npm test
```
